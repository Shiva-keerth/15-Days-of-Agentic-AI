A comprehensive, intensive 15-day curriculum focused on building, scaling, and deploying autonomous AI Agents. This repository maps the journey from basic ReAct agents to complex, multi-agent enterprise RAG systems using modern orchestration frameworks.

---

## 🚀 Project Overview

The shift from standard LLMs (chatbots) to **Agentic AI** (systems that can plan, use tools, and execute workflows) is the next frontier of generative AI. 

This repository is structured as a progressive 15-day bootcamp. It covers:
1. **Core Agent Mechanics** (ReAct, Memory, Tool Calling)
2. **Specialized Agents** (Web Research, SQL generation, RAG)
3. **Advanced Orchestration** (LangGraph, Multi-Agent Supervisors, Human-in-the-Loop)
4. **Production & Deployment** (Serving with FastAPI, Agent Evaluation)

---

## 🗓️ 15-Day Curriculum Architecture

### Phase 1: Core Mechanics & Single Agents
- **Day 01:** Basic ReAct (Reason & Act) Agent
- **Day 02:** Conversational Memory & State Management
- **Day 03:** Custom Tool Calling (APIs, Calculators)
- **Day 04:** Web Research Agent (Tavily/SerpAPI Integration)
- **Day 05:** SQL Database Agent (Text-to-SQL Execution)

### Phase 2: RAG & Reflective Systems
- **Day 06:** Document Q&A & Basic RAG Agent
- **Day 07:** Advanced RAG (Hybrid Search & Re-ranking)
- **Day 08:** Self-Reflective Agent (Self-Correction & Critique Loops)

### Phase 3: LangGraph & Multi-Agent Orchestration
- **Day 09:** LangGraph Basics (Nodes, Edges, State)
- **Day 10:** Human-in-the-Loop (Interrupting & Approving Actions)
- **Day 11:** Multi-Agent Supervisor Routing
- **Day 12:** Multi-Agent Collaboration (Hierarchical Workflows)

### Phase 4: Productionization
- **Day 13:** Agent Serving (Deploying as REST APIs via FastAPI)
- **Day 14:** Agent Evaluation (Benchmarking Tool Accuracy)
- **Day 15:** Enterprise Agentic RAG (The Final Capstone)
